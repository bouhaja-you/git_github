Initialiser un Dépôt

Apprenez à transformer un dossier de projet ordinaire en un dépôt Git capable de suivre les versions de vos fichiers.

lightbulb

Concept Clé

La commande git init crée un nouveau sous-dossier caché nommé .git. C'est ici que Git stocke toutes les métadonnées de suivi.

$ git init
